# Tabuada - Curso Fullstack Master

Este projeto foi construído no curso Fullstack Master do [DevPleno](https://devpleno.com).

## Início


### Pré-requisitos:

Você precisa do NodeJS e do NPM ou YARN instalado em sua máquina.

```
npm install
npm run dev
```

Caso use o YARN

```
yarn add
yarn start
```
